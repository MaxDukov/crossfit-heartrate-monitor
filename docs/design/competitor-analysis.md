## 2. Модели монетизации

| Модель | Примеры | Описание |
|---|---|---|
| **Подписка: устройство + ПО** | WHOOP ($199–$359/год), Polar, Myzone | Устройство включено или продано, ПО по подписке |
| **Per-athlete / per-seat** | SugarWOD, PushPress Train, BTWB | Масштабируется с числом атлетов в зале |
| **Клубное членство / франшиза** | Orangetheory | ПО встроено в стоимость тренировки |
| **Freemium + e-commerce** | Keep, Codoon | Бесплатное приложение + платный контент + устройства |

**Тренд 2025–2026 (BYOD):** Myzone Go, Polar 360 — участники подключают свои Apple/Garmin/Polar-часы вместо покупки отдельного поясного датчика. Снижает барьер входа.

## 3. Анализ графического интерфейса — карточки спортсменов

### 3.1. Раскладка экрана

| Приложение | Раскладка | Масштабируемость |
|---|---|---|
| **Myzone** | Grid из Live Tiles (1 на участника) | Весь зал одновременно |
| **Orangetheory** | Список/сетка плиток, цвет-бары | Скролл для больших классов |
| **BTWB Workout TV** | Multi-mode: лидерборд / pacing / PR / check-in | Настраиваемая плотность: от 20 до 200 атлетов, 32"–70" |
| **SugarWOD BoxTV** | Лидерборд (ранжированный) + whiteboard | По результатам WOD |
| **Наше приложение** | Адаптивный grid (1×1 → 4×2) | Container Queries (cqw), 1–8 атлетов |

### 3.2. Данные на карточке

| Данные | Myzone | Orangetheory | BTWB | Мы |
|---|---|---|---|---|
| Имя | Да | Да | Да | Да |
| ЧСС (bpm) | % max HR | bpm + % | Результат WOD | bpm |
| Зона (цвет) | Вся плитка | Вся строка | — | Цвет цифр + градиент |
| Баллы/очки | MEPs | Splat Points | PR-бейджи | — |
| Калории | Да | Да | — | — |
| График HR | — | — | — | Да (15 мин) |
| % от максимума | Да | Да | — | Да |

### 3.3. Цветовые решения

| Приложение | Палитра | Принцип |
|---|---|---|
| **Myzone** | Grey → Blue → Green → Yellow → Red | Вся плитка заливается цветом зоны |
| **Orangetheory** | Gray → Blue → Green → Orange → Red | Вся строка заливается, ярко/насыщенно |
| **BTWB** | Брендовые цвета зала | Кастомизируемые акценты |
| **Мы** | Green → Amber → Red (+ Dark Red 180–220) | Градиент фона + цвет цифр |

**Стандарт индустрии:** 5 зон (Grey→Red). Мы используем 4 — проще для восприятия, но缺乏 визуального разделения восстановления (grey) и разминки (blue).

## 4. Сильные и слабые стороны нашего приложения

### Уже лучше конкурентов:
- **График истории пульса** (15 мин) на каждой карточке — нет ни у Myzone, ни у Orangetheory
- **WoD-генератор** с базой из 54 шаблонов + фильтр по инвентарю
- **Таймер** 4 типов прямо на экране
- **4-зонная модель** вместо 5 — проще для восприятия
- **Container Queries** — адаптивные карточки (конкуренты используют фиксированные размеры)
- **ANT+ 8 каналов** на один стик — без покупки проприетарных датчиков

### Отстаём:
- Нет геймификации (баллы усилий, бейджи, статусы)
- Фон карточки не заливается цветом зоны
- Нет расчёта калорий
- Нет мобильного приложения для спортсменов
- Нет интеграций со Strava/Apple Health/Google Fit
- Нет аналитики для владельца зала (at-risk detection, тренды)
- Нет социального слоя (лидерборды между сессиями, достижения)
